## Texte enrichi

Vous pouvez utiliser:
- `backquote` pour du code inline
- *simple astérisque* pour l'italique
- **double astérisque** pour mettre en évidence
- _underscore_ pour souligner
- ~~double tilde~~ pour barrer un texte

## Listes

### Liste à puces
- Premier élément
- Deuxième élément
  - Sous-élément
  - Sous-élément
- Troisième élément

### Liste ordonnée
1. Première étape
2. Deuxième étape
   1. Sous-étape 2.1
   2. Sous-étape 2.2
3. Troisième étape

## Code avec Coloration Syntaxique

### JavaScript
```js
function cg(name) {
    console.log(`Ci-gît ${name}!`);
    return true;
}
cg("ma petite fabrique de trampoline");
// ref à MC Solaar dans Eksassaute évidémment
```
cette mise en forme supporte de nombreux mots-clefs tels que:
py, **php**, html, css, js, **json**, **sql**, **bash**, **diff**
